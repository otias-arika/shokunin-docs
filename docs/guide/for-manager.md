# 現場監督・管理者向けガイド

## 管理者でできること

| 機能 | 説明 |
|------|------|
| 作業手配の作成・LINE送信 | 職人さんへの作業指示をデジタルで送信 |
| 職人さんの承認・却下管理 | 新規登録した職人さんの参加を管理 |
| 署名の承認・却下 | 工事完了署名を確認・承認 |
| 現場QRポスターの生成 | 現場ごとのQRコードポスターをワンクリックで作成 |
| ダッシュボードで全体把握 | 全現場・全職人の状況を一目で確認 |
| 招待リンクの発行 | 新しい職人さんや管理者を安全に招待 |

## 手配管理の流れ

```
管理者が手配を作成
    ↓
職人さんのLINEに通知（マップ・ナビボタン付き）
    ↓
ステータス管理：下書き → 手配済み → 報告済み → 完了
```

## なぜデジタル手配が必要か

建設現場の手配は今も**電話・FAX・口頭が主流**です。「言った・言わない」のトラブルが絶えません。2024年問題で労働時間規制が厳格化された今、効率的な手配は現場の生命線です。

## 職人さんの登録・承認手順

1. 管理者が**招待リンク**を生成してLINEなどで送付
2. 職人さんがリンクからLINEで登録
3. 管理者画面に「承認待ち」として表示
4. 管理者が**承認または却下**
5. 承認するとLINEで自動通知が届く
