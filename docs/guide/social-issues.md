# 職人.newが解決する社会課題

職人.newは「便利なツール」だけではありません。建設・施設管理・保守点検業界が直面する社会課題に正面から取り組みます。

| 課題 | 職人.newのアプローチ |
|------|-------------------|
| **人手不足** | 徹底的なシンプルな仕組みとAIで効率化。職人さんは技術に集中 |
| **高齢化** | LINEだけで使えるシンプルなUI。スマホが使えればOK |
| **若手離れ** | デジタルネイティブ世代に魅力的な現場環境を提供 |
| **インフラ老朽化** | 位置情報付き報告で維持管理を効率化 |
| **安全管理** | AI危険予知で死亡事故ゼロを目指す |
| **外国人労働者** | 多言語対応で言葉の壁を越える |
| **2024年問題** | 業務効率化で労働時間削減に貢献 |

> 国土交通省が目指す「生産性1.5倍」は、AIによる事務自動化なしには達成できません。
