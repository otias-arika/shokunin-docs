# 職人さん向けガイド

## 職人さんでできること

| 機能 | 説明 |
|------|------|
| LINEで写真を送るだけで報告 | 現場写真をあずさに送ると報告書が自動作成 |
| スマホで署名 | 工事完了時にスマホ画面に直接サインするだけ |
| 危険予知AIアドバイス受信 | 現場の安全に関するAIからのアドバイスを受け取れる |
| 現場マップ・ナビ活用 | 初めての現場もGoogleマップで迷わず到着 |
| 手配通知の受信・報告編集 | 仕事の手配がLINEで届き、そのまま報告書を編集できる |

## 基本の使い方

### 毎日の報告

1. 現場で写真を撮影する（普通のカメラで撮るだけでOK）
2. LINEであずさに写真を送る
3. AIが自動でコメントを生成してくれる
4. 内容を確認・修正して完了

### 工事完了の署名

1. 工事完了後、LINEまたは報告書ページから署名画面を開く
2. スマホの画面に指で直接サインする
3. 署名が保存されてPDFをダウンロードできる

::: tip ポイント
アプリのインストール・登録は不要です。LINEさえ使えれば、今日からすぐに使い始められます。
:::
