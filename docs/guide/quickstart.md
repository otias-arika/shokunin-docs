# クイックスタート

## はじめての方へ：操作はたったの2ステップ

職人.newの基本的な使い方は非常にシンプルです。

### Step 1：LINEで「あずさ」と友達になる

下記のQRコードをLINEで読み取り、あずさと友達追加してください。

> 📱 LINEアプリを開いて、「友達追加」→「QRコード」から読み取ってください

### Step 2：工事を選んで写真をアップロードする

| 操作 | 内容 |
|------|------|
| ① 工事選択 | 手配された工事の中から、対象の現場を選択します |
| ② 写真アップロード | 現場で撮影した写真をLINEに送るだけでOK！ |

これだけで工事写真報告書が自動的に作成されます。

---

## 毎日の作業はこれだけ

```
📷 現場で写真を撮る
    ↓
📱 LINEであずさに写真を送る
    ↓
📄 報告書が自動完成！
```
