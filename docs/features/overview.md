# 機能一覧

職人.newが提供する機能の一覧です。

| 機能 | 対象 | 概要 |
|------|------|------|
| [AI機能（工事写真→報告書）](/features/ai) | 職人・管理者 | 写真からAIが自動で報告書を作成 |
| [手配管理](/features/dispatch) | 管理者 | LINEで作業手配を送信・ステータス管理 |
| [マップ機能](/features/map) | 職人・管理者 | 工事現場の位置をGoogleマップで管理 |
| [カレンダービュー](/features/calendar) | 管理者 | 報告書を月別カレンダーで一覧管理 |
| [署名機能](/features/signature) | 職人・管理者 | スマホで電子署名・GPS記録付き |
| [現場QRポスター](/features/qr-poster) | 管理者 | 現場事務所に貼るだけで参加できるQRコード |
| [招待・ユーザー承認](/features/invitation) | 管理者 | 安全な招待リンクでメンバー管理 |
| [多言語対応](/features/multilingual) | 全員 | 日本語・英語・ベトナム語対応 |
| [i-Reporter連携](/features/i-reporter) | 管理者 | 既存のi-Reporterにシームレスに連携 |
| [報告書ダウンロード](/features/report-download) | 職人・管理者 | 完成した報告書を安全にダウンロード |
