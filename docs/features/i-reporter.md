# i-Reporter連携

## 概要

建設業向け帳票アプリ「i-Reporter」とシームレスに連携します。既存の業務フローを壊さずにそのまま使えます。

## 主な機能

- **自動マッピング** — 1シート6枚レイアウトで、工事情報をフォームに自動入力
- **オンプレ・クラウド両対応** — どちらのi-Reporterでも対応（API契約が必要）
- **i-Reporterアプリでの後続作業も可能**

## 必要な環境

i-Reporter側でのAPI契約が必要です。詳細はi-Reporter提供元にご確認ください。

## 設計思想

「今ある仕組みを置き換える」のではなく、「**今ある仕組みに乗る**」設計です。現場の抵抗感を最小限に抑えることが、DX成功の最大のカギです。
