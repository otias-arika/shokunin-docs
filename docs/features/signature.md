# 署名機能

## 概要

工事完了時に、スマホ画面で直接手書き署名ができます。紙の署名に代わる、信頼性の高い電子署名機能です。

## 主な機能

- **手書き署名** — スマホの画面に指で直接サイン
- **GPS位置情報の同時記録** — 「どこで」署名したかが自動記録されます
- **管理者承認フロー** — 管理者による承認・却下をON/OFFで設定可能
- **i-Reporter自動連携** — 署名完了でi-Reporterの完了帳票として扱われます

## なぜ電子署名が重要か

「誰が、いつ、どこで完了を確認したか」の証明が求められます。紙の署名は紛失・改ざんのリスクがありますが、デジタル署名はGPS・タイムスタンプ付きで発注者への信頼性が飛躍的に向上します。
