# 複数現場・企業の管理（マルチテナント）

## 概要

複数の現場・複数の企業を、1つのプラットフォームで安全に管理できます。

## 主な機能

- **テナント間の完全データ分離** — 他社・他現場のデータは一切見えません
- **テナント別の設定管理** — LINE Bot・i-Reporter・AIの設定を現場ごとに管理
- **機密情報の暗号化保存** — 接続情報などの機密データは暗号化して保管
- **サービス接続ステータスの監視** — 各連携サービスの接続状況をリアルタイムで確認

## こんな使い方ができます

| 規模 | 活用例 |
|------|--------|
| 一人親方 | 自分の現場を1テナントで管理 |
| 中小建設会社 | 複数の現場を1社のテナントで一括管理 |
| 大手ゼネコン | 現場・協力会社ごとにテナントを分けて管理 |

## なぜマルチテナントが重要か

建設業の「重層下請構造」に対応しています。一人親方から大手ゼネコンまで、規模を問わず同じ基盤で運用できます。業界全体のDXを底上げします。
