# 招待・ユーザー承認システム

## 概要

暗号化された招待リンクで、管理者や職人さんを安全に招待・管理できます。

## 主な機能

- **安全な招待URL生成** — テナントIDを暗号化した招待URLをワンクリックで生成
- **リンクを送るだけで登録完了** — 難しい操作は不要
- **承認フロー** — LINEから新規登録 → 承認待ち → 管理者が承認/却下
- **自動通知** — 承認・却下時にLINEで自動通知
- **管理者ロールの付与・削除** も可能

## なぜ承認管理が重要か

多くの協力会社が出入りする現場で「誰が関わっているか」を明確にすることは、元請け・発注者からの信頼に直結します。「明日から新しい職人さんが来る」という状況にも、リンクを送るだけで対応できます。
