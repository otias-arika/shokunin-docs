# 多言語対応

## 概要

日本語・英語・ベトナム語に対応しています（ベトナム語は対応中）。外国人職人さんも安心して使えます。

## 対応言語

| 言語 | 状況 |
|------|------|
| 日本語 | ✅ 対応済み |
| 英語 | ✅ 対応済み |
| ベトナム語 | 🔄 対応中 |

## 対応範囲

- UI（画面表示）
- メッセージ
- AIの応答
- LINEユーザー設定から自動言語判定

## なぜ多言語対応が重要か

建設業の外国人労働者は11万人を超えています。「理解できなかった」が重大事故につながります。多言語対応は便利さの問題ではなく、**安全の問題**です。
