# 現場QRポスター

## 概要

現場事務所に貼るだけで、職人さんがQRコードを読み取ってすぐに参加できるポスターを生成します。

## 主な機能

- **ワンクリックでポスター生成** — 管理画面からすぐに作成
- **QRでLINE友達追加 → その場ですぐ使える**
- **印刷して現場事務所に掲示** — テナント（現場）ごとに個別のQRを生成可能
- **毎日の作業報告向け** — 手配ではなく日常的な作業報告に最適

## なぜQRポスターが重要か

建設DX最大の壁は「導入の手間」です。アプリのインストールも登録も不要。60代のベテランも、今日来た外国人も、QRを読むだけで即戦力。**導入障壁ゼロ**を実現します。
