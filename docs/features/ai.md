# AI機能（工事写真→報告書）

## 概要

工事現場で撮影した写真をLINEで送るだけで、AIが自動的に作業記録の下書きを作成します。

## 主な機能

- **AI自動コメント生成** — 写真をAIが解析し、現場に合ったコメントを自動生成します（「写真だけ撮る」現場スタイルそのままでOK）
- **あずさ対話モード** — 建設業に関する知識をあずさに質問・相談できます
- **高度な相談対応** — ウェブ検索と連携した最新情報での回答も可能です

## なぜこれが重要か

職人さんの本分は「技術」。報告書作成に費やす時間をAIが肩代わりすることで、職人さんは本来の仕事に集中できます。

## 使い方

1. 現場で写真を撮影する
2. LINEであずさに写真を送信する
3. AIが写真を解析してコメントを生成
4. 内容を確認・修正して報告書を完成させる
