# 手配管理機能

## 概要

管理者から職人さんへの作業手配をLINEで送信します。電話・FAX・口頭に頼らないデジタルな手配管理を実現します。

## 主な機能

- **手配作成** — 工事名・番号・住所を指定して手配を作成。下書き保存にも対応
- **LINEリッチメッセージ通知** — マップ・ナビボタン付きで職人さんのLINEに届く
- **ステータス管理** — 下書き → 手配済み → 報告済み → 完了 の流れで進捗を把握

## なぜデジタル化が必要か

建設現場の手配は今も電話・FAX・口頭が主流です。「言った・言わない」のトラブルが絶えません。2024年問題で労働時間規制が厳格化された今、効率的な手配は現場の生命線です。
